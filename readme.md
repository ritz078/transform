# transform-www

> Website for https://transform.now.sh

This project is based on [next.js](https://github.com/zeit/next.js/)

## Development

- Fork the repo
- Create a new branch.
- `yarn install`
- `yarn dev` then fix issues or add features.
- Make a pull request.
